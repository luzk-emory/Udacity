Exploration of the Ford GoBike System Data¶

by Zhikun Lu¶

Preliminary Wrangling¶

This dataset is from Lyft Bay Wheel. (source: https://www.lyft.com/bikes/bay-wheels/system-data) It includes information about individual rides made in a bike-sharing system covering the greater San Francisco Bay area. Each trip is anonymized and includes:

  • Start Time and Date
  • End Time and Date
  • Start Station ID
  • Start Station Name
  • Start Station Latitude
  • Start Station Longitude
  • End Station ID
  • End Station Name
  • End Station Latitude
  • End Station Longitude
  • User Type (Subscriber or Customer Subscriber = Member or Customer = Casual)

and other variables.

I am going to explore the following issues with this dataset:

  1. When are most trips taken in terms of time of day, day of the week, or month of the year?
  2. How long does the average trip take?
  3. Does the above depend on if a user is a subscriber or customer?

I manually download 18 months data from the website, from Jan 2019 to Jun 2020. Before I start the exploration, I need first clean and tidy the data:

  • The variable names are not consistent, which I need correct.
  • The raw data have 18 separate tables; I need combine them together.
  • The labels in user_type are not consistent.

What is the structure of your dataset?¶

The data contains the following variables

  • start_time: starting time of the trip
  • end_time: ending time of the trip
  • start_lat: latitude of the starting location
  • start_lng: longitude of the starting location
  • end_lat: latitude of the ending location
  • end_lng: longitude of the ending location
  • user_type: type of user, either subscriber or non-subscriber

What is/are the main feature(s) of interest in your dataset?¶

The dataset contains detailed geographic information for each trip, i.e., the starting location and ending location. Also, the trip start time and end time can provide rich information on travel time.

What features in the dataset do you think will help support your investigation into your feature(s) of interest?¶

I am gonna explore the following three questions with this dataset:

  1. When are most trips taken in terms of time of day, day of the week, or month of the year?
  2. How long does the average trip take?
  3. Does the above depend on if a user is a subscriber or customer?

Features such as start_time, end_time, start_lat, start_lng, end_lat, and end_lng will help me directly address these questions.
